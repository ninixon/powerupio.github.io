GNU/Linux

Distributions

Debian
One of the earliest Linux distributions. Stable, Testing, and Unstable releases.
Ubuntu
Commercially sponsored, Debian-derived. 6 month release cycle. Server Editions.
Mint
Debian/Ubuntu-based. Full multimedia support out of the box. 6 month release cycle.
Red Hat
Red Hat Enterprise Linux is targeted toward the commercial market.
Fedora
Sponsored by Red Hat.
6 month release cycle.
Server Edition.
CentOS
A free rebuild of source packages from the Red Hat Enterprise Linux. 2 year release cycle.
openSUSE
User-friendly desktop, tools for developers and sysadmins. 12 month release cycle.
PCLinuxOS
Aimed at ease of use. Script which allows hard drive snapshots to ISO. Rolling Release.
Mageia
Stable and trustable governance to direct collaborative projects. 9 month release cycle.
Arch
Focuses on simplicity of design. Installed without a graphical front-end. Rolling release.
Gentoo
Software is compiled locally from source code. Rolling release.
Slackware
The original Linux distribution, geared towards power and stability.
Puppy
Lightweight, small installation. The entire system can be run from RAM.
Dragora
GNU/Linux-Libre. Simple packaging system. Endorsed by the FSF.
Android-x86
Project to port Android x86 platform.

More at www.distrowatch.com.
